About

Example website using JS, React, and MobX

Install

Install Node

Run package installer

    npm install npm -g
    npm install

Then copy the contents of the "custom docdash" folder into the folder 'app/node_modules/docdash' and replace contents. This will set up our custom styling templates for code documentation.

Locally Load Website

From your terminal run:

    npm start

Additional Scripts

For Doc Generating

Ensure the following is in one of your bash profiles (if node packages not found when running)

    export PATH="{PATH_TO_REPO}/app/node_modules/.bin:$PATH"
    npm run doc

For testing:

    npm run test

Testing will also run prior to pushing to a remote server.

For Linting:

    npm run lint

Linting will also run prior to commiting and/or pushing to a remote server.

Building:

    npm run build
